## 대회 공동 규칙

- **제출 횟수 제한** : <u>1시간에 1회</u>
- **사용 가능 언어** : <u>Python</u>
- **대회별 문의** : 각 대회별 `Repositoy`에서 `Issues`에 글을 남김
- **추론 시간** : 1시간 이내
  ### 실격 사유

  **아래에 해당하는 모든 사유는 대회 실격 사유에 해당됩니다.**

  - 모델 학습 과정에서 검증 혹은 평가 데이터셋 활용(Data Leakage)
  - 답안 수기 작성
  - 기타 공정한 경쟁을 저해하는 행위
